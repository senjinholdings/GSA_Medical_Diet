# テスト計測について

## 手順

1. **赤字の文字列が含まれているパラメータ付きURL** にてテスト計測をお願いいたします。  
   例: `https://eminal-clinic.jp/?refmedia=xxxxx`

2. テスト時に入力したメールアドレス（`afi_monitoring@orizo.co.jp`）は  
   オリゾの林田尚樹、上田愛美宛に届きます。

3. 完了画面にて **【ご予約番号】** をコピーし、林田もしくは上田までご連絡ください。  
   キャンセル処理をいたします。

   - ご予約番号の例: `202405171319_2b9bafbbbe`

4. 新規原稿による配信開始をお願いいたします。
