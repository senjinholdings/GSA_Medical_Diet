# マウスピース矯正サイト E2Eテストスイート

このプロジェクトは、マウスピース矯正サイトの地域名表示機能に対する包括的なE2Eテストスイートです。

## 概要

マウスピース矯正サイトでは、URLパラメータ `region_id` に基づいて各セクションの地域名が動的に変更される機能があります。このテストスイートは、その機能が正しく動作することを確認します。

## テスト対象機能

### 1. 地域名表示機能
- **MVセクション**: `#mv-region-name` と `#mv-region-text` での地域名表示
- **ランキングセクション**: `#rank-region-name` での地域名表示  
- **比較表セクション**: `#comparison-region-name` での地域名表示
- **詳細バナーセクション**: `#detail-region-name` での地域名表示

### 2. 無効値ハンドリング
- `region_id=000`, `region_id=0`, `region_id=999`, `region_id=invalid` などの無効な値
- 無効な値の場合は東京（`region_id=013`）にフォールバックする

### 3. 地域セレクター機能
- `#sidebar-region-select` の存在確認
- ラベルとの適切な関連付け確認

### 4. 詳細バナー機能
- 地域名と件数の動的表示確認
- `#detail-rank-count` での件数表示

## セットアップ

### 必要な依存関係のインストール

```bash
npm install
```

### サーバーの起動

テスト実行前にローカルサーバーを起動：

```bash
npm start
# または
node server.js
```

サーバーは http://localhost:8090 で起動します。

## テスト実行

### 全テストの実行

```bash
npm test
# または
npx playwright test
```

### 特定のテストファイルのみ実行

```bash
# 包括的テストスイート
npx playwright test tests/mouthpiece-final.spec.js

# 簡略版テスト
npx playwright test tests/mouthpiece-e2e-simplified.spec.js

# マウスピース関連テストのみ
npm run test:mouthpiece
```

### ヘッド付きモードでの実行

```bash
npm run test:headed
# または
npx playwright test --headed
```

### デバッグモード

```bash
npm run test:debug
# または
npx playwright test --debug
```

## テストレポート

テスト実行後にHTMLレポートを表示：

```bash
npm run test:report

npx playwright show-report
```

## 対応地域

テストで使用される主要な地域ID：
- `013`: 東京（デフォルト）
- `027`: 大阪
- `001`: 北海道
- `000`: 全国（無効値として扱われ、東京にフォールバック）

## 注意事項

1. **データ読み込み**: JavaScriptでの動的データ読み込みのため、テストでは適切な待機時間を設けています
2. **404エラー**: 一部のリソース（画像等）で404エラーが発生しますが、主要機能には影響しません
3. **タイムアウト**: 重い処理がある場合は、テストタイムアウトを調整可能です

## トラブルシューティング

### サーバーが起動しない場合
```bash
# ポート8090が使用中の場合、別のポートを使用
node server.js
```

### テストが失敗する場合
1. サーバーが正しく起動しているか確認
2. ブラウザーが正しくインストールされているか確認：
   ```bash
   npx playwright install
   ```

### デバッグ情報の取得
テスト実行時に詳細なログが出力されるため、失敗原因を特定できます。
